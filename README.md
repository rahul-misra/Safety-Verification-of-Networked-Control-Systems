Repository for the paper "Safety Verification of Networked Control Systems via Barrier Certificates for Piecewise Deterministic Markov Processes" authored by Sina Hassani, Manuela L. Bujorianu, Rahul Misra, and Rafal Wisniewski.
Purpose - Generate a Stochastic Barrier certificate for a PDMP representing an IEEE 39 bus networked control system using Sum-of-Squares.
Dependencies - 1) MATLAB; 2) YALMIP; 3) MOSEK
